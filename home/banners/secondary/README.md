# Secondary Homepage Carousel

Place the carousel images that appear below the six project icons in this folder.

Use numeric file names in display order:

- `1.png`
- `2.png`
- `3.png`
- `4.png`

The secondary carousel currently expects:

`/assets/home/banners/secondary/1.png`

`/assets/home/banners/secondary/2.png`

`/assets/home/banners/secondary/3.png`

When adding more images later, keep numbering them in order and add the matching item in `config.js`.

Manage each image's title, subtitle, detail, and H5 URL in:

`/assets/home/banners/secondary/config.js`
