# Home Banner Assets

Put homepage carousel images here.

- `top/`: first screen carousel shown at the top of the home page.
- `secondary/`: carousel shown below the six project icons.

Recommended image ratios:

- Top banner: keep a consistent wide ratio, for example 16:9 or 3:2.
- Secondary banner: use the same ratio as the top banner unless a different layout is intentionally designed.

The current home page uses a fixed banner height in WXSS, so images are scaled to fit inside the slot rather than changing the slot height.
